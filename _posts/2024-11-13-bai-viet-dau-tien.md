Đây là bài viết hướng dẫn cách sử dụng Jekyll để tạo blog trên GitHub Pages. Trong bài viết này, bạn sẽ học cách:

- Cài đặt Jekyll
- Tạo và cấu hình blog trên GitHub Pages
- Sử dụng các theme Jekyll

## Bước 1: Cài đặt Jekyll

Để cài đặt Jekyll, bạn cần cài đặt Ruby và Bundler trước. Sau khi cài đặt xong, chạy lệnh sau:

```bash
gem install jekyll bundler
```
